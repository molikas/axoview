# About

Axoview is an open-source isometric diagramming library, released under the MIT license. It is forked from [FossFLOW](https://github.com/stan-smith/FossFLOW) — itself a fork of [Isoflow](https://github.com/markmanx/isoflow) — and ships the same `@isoflow/isopacks` icon collections (AWS / GCP / Azure / Kubernetes / core), which remain attributed to the Isoflow project.

Source: [github.com/molikas/axoview](https://github.com/molikas/axoview).
