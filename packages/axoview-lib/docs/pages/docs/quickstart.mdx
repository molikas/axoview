# Quick Start

Axoview can be imported as an ES6 module:

```jsx
import Axoview from "axoview";
```

### Basic usage

```jsx showLineNumbers
import React from 'react';
import Axoview from 'axoview';

const App = () => {
  return (
    <Axoview />
  );
}

export default App;
```

**Note**: this will display a blank Axoview editor, without icons (which is not very useful!).  To initialise the editor with an iconset, see [Loading Isopacks](/docs/isopacks).

### Dimensions of Axoview

Axoview takes _100%_ of `width` and `height` of the containing block so make sure the container in which you render Axoview in has non-zero dimensions.

### Integration with NextJS

Axoview cannot be server-side rendered and has to be imported using `next/dynamic`:

```jsx showLineNumbers filename="IsoflowDynamic.jsx"
import dynamic from 'next/dynamic';

export const IsoflowDynamic = dynamic(() => {
    return import('axoview');
  },
  {
    ssr: false
  }
);
```

```jsx showLineNumbers filename="App.jsx"
import { IsoflowDynamic } from './IsoflowDynamic';

const App = () => {
  return (
    <IsoflowDynamic />
  );
}

export default App;
```
